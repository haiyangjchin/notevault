# LLM Wiki 知识库

## 项目概述

基于 Karpathy 的 Wiki 方法论，结合 Obsidian + Claudian 搭建的个人 AI 知识管理系统。

## 目标

1. 建立结构化的个人知识库
2. 用 LLM 作为智能检索和整理层
3. 实现知识的自动关联和健康检查
4. 形成可持续的知识积累习惯

## 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 笔记工具 | Obsidian | 主力知识管理平台 |
| AI 接口 | Claudian | Obsidian 中的 Claude Code 图形界面 |
| 查询引擎 | Dataview | 结构化查询笔记元数据 |
| AI 模型 | Claude | 知识整理和关联发现 |
